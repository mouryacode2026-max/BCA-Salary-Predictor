# BCA-Salary-Predictor
 ​🤖 Salary Predictor AI (FastAPI + Machine Learning)
​A full-stack Machine Learning application that predicts a professional's salary based on their Age, Years of Experience, and Skill Score. This project was developed as part of my continuous learning in AI and Full-Stack development.
​🚀 Key Features
​Machine Learning Model: Uses a Linear Regression model built with Scikit-Learn.
​FastAPI Backend: A modern, high-performance Python web framework for the API.
​Interactive UI: A clean HTML/CSS frontend that allows users to input data and see real-time predictions.
​JSON Integration: Seamless communication between the frontend and the AI model.
​🛠️ Technology Stack
​Programming Language: Python
​AI/ML Libraries: Scikit-Learn, NumPy
​Backend Framework: FastAPI
​Server: Uvicorn
​Frontend: HTML5, CSS3, JavaScript (Fetch API)
​📂 File Structure
​main.py — The core FastAPI application and prediction logic.
​model.pkl — The saved Machine Learning model.
​index.html — The user interface for entering data.
​README.md — Project documentation.
​💻 How to Run the Project
1.Clone the repository:
git clone https://github.com/YOUR_GITHUB_USERNAME/YOUR_REPO_NAME.git
2.Install the required libraries:
pip install fastapi uvicorn scikit-learn numpy
3.Start the server:
uvicorn main:app --reload
4.View the App:
Open your browser and navigate to http://127.0.0.1:8000
